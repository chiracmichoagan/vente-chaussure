# Sole & Style — Site Vitrine Chaussures

Site vitrine e-commerce pour un vendeur de chaussures, construit avec TanStack Start, React et Tailwind CSS, déployé sur Netlify.

## Technologies utilisées

- **TanStack Start** — framework React full-stack avec routing typé
- **React 19** — UI
- **Tailwind CSS v4** — styles utilitaires
- **Netlify** — hébergement et fonctions serverless

## Fonctionnalités

- Page d'accueil avec hero, statistiques, catégories, catalogue filtrable et témoignages
- Catalogue de 6 modèles de chaussures avec filtres par catégorie
- Page produit détaillée : sélection de pointure, coloris, produits similaires
- Header sticky et footer avec infos de contact
- Design responsive mobile-first

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
